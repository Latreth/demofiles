Exemple Javascript

Ce document contient le code Javascript suivant : 

<script type="text/javascript">
   function essai() {
	i = 0;
	if ( i < 1 ) {
		alert('Hello');
	}
   }
</script>

qui se déclenche à l'aide de cet hyperlien

Malheureusement, ce code contient un caractère < qui est interprété par un parseur XML et de ce fait ce document n'est pas valide, puisqu'il n'est même pas bien formé au sens de XML.

Ceci peut se vérifier en tentant de valider ce document par exemple auprès du W3C (cf. logo en bas de page)
