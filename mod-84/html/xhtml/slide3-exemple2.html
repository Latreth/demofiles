Section CDATA

Exemple Javascript

Ce document contient le code Javascript suivant : 

<script type="text/javascript">
 <![CDATA[
   function essai() {
	i = 0;
	if ( i < 1 ) {
		alert('Hello');
	}
   }
 ]]>
</script>

qui devrait logiquement se déclencher à l'aide de cet hyperlien.

Ce document est conforme aux recommandations, ce qui peut se vérifier en le validant auprès du W3C (cf. logo en bas de page).

Malheureusement, ce petit programme ne fonctionne pas, puisque l'interpréteur des navigateurs bute sur la syntaxe de la balise CDATA en tentant de l'analyser comme du code Javascript (ce qui peut se vérifier pour les plus geeks d'entre vous à l'aide de la console, via les outils développeur de votre navigateur).
