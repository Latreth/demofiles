Section CDATA

Exemple Javascript

Ce document contient le code Javascript suivant : 

<script type="text/javascript">
 //<![CDATA[
   function essai() {
	i = 0;
	if ( i < 1 ) {
		alert('Hello');
	}
   }
 //]]>
</script>

qui fonctionne comme attendu lorsqu'on le déclenche à l'aide de cet hyperlien.

En effet, la balise CDATA est cette fois-ci protégée par un commentaire Javascript, ce qui évite aux navigateurs de l'interpréter.

Ce document est conforme aux recommandations, ce qui peut se vérifier en le validant auprès du W3C (cf. logo en bas de page).
