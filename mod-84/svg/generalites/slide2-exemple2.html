Scalable Vector Graphics

Dimensions initiales du graphique

Cette page affiche divers documents SVG, chaque fois sans préciser la dimension de l'image. Ces documents diffèrent uniquement par la valeur des attributs WIDTH et HEIGHT de l'élément principal svg.

Vous devriez voir apparaître trois fois la même image, avec à chaque fois une taille différente.

width="180"
height="180"
(x1)
width="120"
height="120"
(x2/3)
width="90"
height="90"
(x1/2)

L'élément SVG racine de chacun des graphiques est du type : 

 <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 180 180" width="180" height="180">
 ...
 </svg>
Le code HTML utilisé pour afficher chacune de ces images : 

 <object type="image/svg+xml" data="slide2-exemple2-1.svg" ></object>

 <object type="image/svg+xml" data="slide3-exemple2-2.svg" ></object>

 <object type="image/svg+xml" data="slide3-exemple2-3.svg" ></object>
