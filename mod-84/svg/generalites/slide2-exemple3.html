Scalable Vector Graphics

Dimensions initiales du graphique

Cette page affiche un même document SVG, en modifiant à chaque fois la dimension de l'image à l'aide des attributs WIDTH et HEIGHT de la balise OBJECT.

Vous devriez voir apparaître trois fois la même image, avec à chaque fois une taille différente.

width="180"
height="180"
(x1)
width="120"
height="120"
(x2/3)
width="90"
height="90"
(x1/2)

Le code HTML utilisé pour afficher les images : 

 <object type="image/svg+xml" data="slide3-exemple1.svg" width="180" height="180" ></object>

 <object type="image/svg+xml" data="slide3-exemple1.svg" width="120" height="120" ></object>
 
 <object type="image/svg+xml" data="slide3-exemple1.svg" width="90" height="90" ></object>
