Scalable Vector Graphics

Redimensionnement par le navigateur

Cette page affiche deux documents SVG, en modifiant le rapport initial hauteur / largeur; à l'aide des attributs WIDTH et HEIGHT de la balise OBJECT.

Vous devriez voir apparaître deux fois la même image, la première déformée, la seconde non.

width="240"
height="90"
width="240"
height="90"

Le code HTML utilisé pour afficher les deux images (au nom de l'image près) : 

 <object type="image/svg+xml" data="slide3-exemple1.svg" width="240" height="90" >
 </object>

Et le code SVG permettant de ne pas redimensionner l'image : 

 <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 180 180" preserveAspectRatio="none">
 ...
 </svg>
