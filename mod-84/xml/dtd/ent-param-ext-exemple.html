Entité paramètre externe

La définition ci-dessous déclare et instancie l'entité paramètre xhtml-lat1.ent afin de bénéficier de la déclaration de toutes les entités du type caractères accentués, classiques employés en HTML : 

<!DOCTYPE classe  [

	<!ELEMENT classe (étudiant+)>
	<!ATTLIST classe option CDATA #FIXED "TIC">

	<!ELEMENT étudiant (prénom,nom)>
	<!ELEMENT prénom (#PCDATA)>
	<!ELEMENT nom (#PCDATA)>

	<!ENTITY % HTMLlat1 PUBLIC
	    "-//W3C//ENTITIES Latin 1 for XHTML//EN" "xhtml-lat1.ent">

	%HTMLlat1;
]>

Ces entités peuvent donc être librement employées au sein du document : 

 <classe>
  <étudiant>
   <prénom>B&eacute;n&eacute;dicte</prénom>
   <nom>Tinne</nom>
  </étudiant>
  <étudiant>
   <prénom>Jean-Fran&ccedil;ois</prénom>
   <nom>P&egrave;re-No&euml;l</nom>
  </étudiant>
 </classe>
