Exemple XML

Entité Générale

Soit le document XML ci-dessous : 

  <?xml version="1.0" encoding="ISO-8859-1"?>

  <!DOCTYPE essai [
	<!ELEMENT essai (#PCDATA)>
  ]>

  <essai>
   Ceci est un essai de caractère accentu&eacute; !
  </essai>

Voici le résultat d'une tentative de validation : 


Fatal Error at file /~muller/cours/xml/dtd/ent-general-exemple1-1.xml, line 8, char 50
  Message: Entity 'eacute' was not found

Essayer de valider le document en ligne - Essayer de voir le document dans votre navigateur

En déclarant correctement l'entité eacute : 

  <?xml version="1.0" encoding="ISO-8859-1"?>

  <!DOCTYPE essai [
	<!ELEMENT essai (#PCDATA)>
	<!ENTITY eacute "&#233;">
  ]>

  <essai>
   Ceci est un essai de caractère accentu&eacute; !
  </essai>

le document est valide : 

ent-general-exemple1-2.xml: 5 ms (1 elems, 0 attrs, 0 spaces, 46 chars)
