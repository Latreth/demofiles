Exemple XML

Entité Générale

Voici document XML comportant une déclaration d'entité : 

  <?xml version="1.0" encoding="ISO-8859-1"?>

  <!DOCTYPE essai [
	<!ELEMENT essai (#PCDATA|br)*>
	<!ELEMENT br EMPTY>
	<!ENTITY contact "Daniel.Muller@ec-lyon.fr">
  ]>

  <essai>
   Ceci est un document XML comportant une déclaration d'entité.
   <br/>
   Contact : &contact;
  </essai>
