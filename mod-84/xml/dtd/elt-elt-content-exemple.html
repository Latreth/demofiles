Modèles de contenu élémentaire

L'élément étudiant déclaré ci-dessous devra comporter un nom et un ou plusieurs prénom, dans n'importe quel ordre : 

 <!DOCTYPE classe [
	<!ELEMENT classe (promo,étudiant+)>
	<!ELEMENT étudiant ((nom,prénom+)|(prénom+,nom,prénom*))>

	<!ELEMENT promo  (#PCDATA)>
	<!ELEMENT nom    (#PCDATA)>
	<!ELEMENT prénom (#PCDATA)>
 ]>

Le document ci-dessous est conforme à cette DTD : 

 <classe>
  <promo>1901</promo>
 
  <étudiant>
   <nom>Deubaze</nom>
   <prénom>Raymond</prénom>
  </étudiant>
 
  <étudiant>
   <prénom>Betty</prénom>
   <prénom>Barbara</prénom>
   <prénom>Noëlle</prénom>
   <prénom>Candide</prénom>
   <prénom>Sandra</prénom>
   <nom>Duciel</nom>
  </étudiant>
 
  <étudiant>
   <prénom>Artémise</prénom>
   <nom>Fenouillard</nom>
   <prénom>Cunégonde</prénom>
  </étudiant>
 
 </classe>
