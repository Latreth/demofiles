1.5 Code relatif à 2n-1

Principe

En code relatif à 2n-1, tous les nombres (positifs et negatifs), sont représentés par : 

Code(x) = 2n-1 + x
Exemple

La droite 4 bits donne les codes des nombres -8 à +7 : 

    -8                  -1     0    +1                  +7   
-----+----- - - - - -----+-----+-----+----- - - - - -----+-->
   0000                0111  1000  1001                1111
On peut remarquer que ce code est identique au code complément à 2, avec le bit de signe complémenté. Ceci peut se démontrer :

x > 0 : Code(x) = 2n-1 + x = Code2(x) + "bit de signe"

x < 0 : Code(x) = 2n-1 + x = 2n-1 - |x| = 2n - |x| - 2n-1 = Code2(x) - "bit de signe"

 

Addition

L'addition de deux nombres est effectuée comme en code complément à 2, en sommant simplement les codes des opérandes.

La soustraction est obtenue par addition de l'opposé.

L'opposé d'un nombre se calcule par : 
            _________    
Code(x<0) = Code(|x|) + 1
Le dépassement est géré comme en code complément à 2.
